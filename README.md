# 💰 FinSight Dashboard

A clean and interactive **Finance Dashboard UI** built to help users track transactions, analyze spending patterns, and gain meaningful financial insights.

🌐 **Live Demo:** https://fin-sight-dashboard-theta.vercel.app/
📂 **GitHub Repository:** https://github.com/Hanumantuphade/fin-sight-dashboard

---

## 🚀 Features

### 📊 Dashboard Overview

* Summary cards for **Total Balance, Income, and Expenses**
* Time-based visualization for financial trends
* Category-wise spending breakdown

### 💳 Transactions

* List of transactions with:

  * Date
  * Amount
  * Category
  * Type (Income / Expense)
* Features:

  * 🔍 Search functionality
  * 🔽 Filtering
  * ↕️ Sorting

### 👤 Role-Based UI

* **Viewer:** Can only view data
* **Admin:** Can add/edit transactions
* Role switching using dropdown (frontend simulation)

### 📈 Insights

* Highest spending category
* Monthly comparison
* Useful financial observations

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/Hanumantuphade/fin-sight-dashboard.git

# Navigate into the project
cd fin-sight-dashboard

# Install dependencies
npm install

# Start development server
npm run dev
```
